# Web Article Sentiment Analyzer
This project analyzes the sentiment of web articles using Natural Language Processing (NLP).

# Tools & Technologies
- Python
- Pandas
- NLP (Text Cleaning, Stopwords, Sentiment Analysis)

# Description
The project extracts text from web articles, preprocesses the data, and calculates sentiment scores
using positive and negative word dictionaries.

# Output
The final output is generated in Excel format containing sentiment scores.
